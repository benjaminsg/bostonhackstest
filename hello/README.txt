* Quickstart guide: http://flask.pocoo.org/docs/1.0/quickstart/#quickstart

* How to run:

> export FLASK_APP=hello.py

on UNIX/OSX

> set FLASK_APP=hello.py

on Windows

> flask run

or

> export FLASK_APP=hello.py
> python -m flask run

* Open this link in browser to view site:
http://127.0.0.1:5000/


* To make the server public, make sure debugger is disabled and run:

> flask run --host=0.0.0.0
